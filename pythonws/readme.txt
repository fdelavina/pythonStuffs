pruebas de servidor http y websockets (cliente y servidor).
crear primero un entonro virtual con la version de python
que se tenga instalada en el pc (en mi caso 3.9.0)
>>> python -m venv my_env
entrar en el entorno virtual
>>> /bin/Scripts/activate.exe
>>> instalar las dependencias del archivo requeriments.txt con pip

>>> pip -m install requeriments.txt

ejecutar cualquiera de los scripts
>>> python script.py
